Nested Lists

  • Used to add a list within a list
  • Ordered Lists and Unordered lists can
    have different levels.
    •    -To display different levels of hierarchy
        (subcategories)
  • Each Will be indented appropriately by
    your web browser
    •    -Browsers will usually apply a different style to
        the nested.(example: Disc for main
        category and open circle for subcategory.)
